# 楽天ブログ投稿ヘルパー 初期版メモ

作業フォルダ内に、Chrome 拡張機能として読み込める初期版を作成しました。

## 実装済み

- 記事の作成、保存、編集、複製
- 記事状態の管理
- テンプレートの保存と反映
- PR 表記、定型文、アフィリエイトリンクを含む本文合成
- 転送前チェック
- 簡易プレビュー
- JSON バックアップと復元
- 楽天ブログ投稿画面の起動
- タイトルと本文の自動入力
- 下書き保存ボタンが見つかる場合のみ自動クリックを試行

## まだ調整が必要な点

- 楽天ブログの実際の投稿画面セレクタ確認
- カテゴリ、テーマ、投稿日時の確実な自動選択
- リッチテキストエディタ固有の入力方法
- 実サイト上での下書き保存可否の検証

## 主要ファイル

- `rakuten-blog-helper/app.html`
- `rakuten-blog-helper/app.js`
- `rakuten-blog-helper/content-script.js`
- `rakuten-blog-helper/background.js`
- `rakuten-blog-helper/manifest.json`
