# 楽天ブログ投稿ヘルパー

楽天ブログの記事作成、ローカル保存、テンプレート管理、JSONバックアップ、楽天ブログ投稿画面への安全な転送を行う Chrome 拡張機能です。

## 初期版でできること

- 記事タイトル、本文、カテゴリ、テーマ、PR表記、定型文、アフィリエイトリンクなどの保存
- 記事一覧、再編集、複製、状態管理
- テンプレート保存と記事への反映
- 転送前チェックと簡易プレビュー
- 全データの JSON エクスポートとインポート
- 楽天ブログ投稿画面を開いてタイトルと本文を自動入力
- 下書き保存ボタンが見つかる場合のみ自動クリックを試行

## インストール

1. Chrome 系ブラウザで `chrome://extensions` を開く
2. デベロッパーモードを有効化する
3. `パッケージ化されていない拡張機能を読み込む` を押す
4. この `rakuten-blog-helper` フォルダを選ぶ

## 使い方

1. 拡張機能アイコンから `作成ツールを開く` を押す
2. 記事タイトルと本文を入力し、必要なら PR 表記やテンプレートを設定する
3. `記事を保存` でローカル保存する
4. 楽天ブログの新規投稿 URL が分かる場合は `転送設定` に保存する
5. `楽天ブログへ転送` を押す
6. 楽天ブログ側で内容を確認し、必要に応じて手動で下書き保存または公開する

## 注意点

- 楽天ブログに正式 API がない前提のため、画面の HTML 構造変更で自動入力が失敗する可能性があります
- `content-script.js` の `SITE_PROFILE` に楽天ブログ固有のセレクタを集約しています
- ログイン情報やパスワードは保存しません
- ブラウザのサイトデータ削除で保存内容が消える可能性があるため、定期的に JSON バックアップを取ってください
